## Decision making statements
Based on a condition, if you want to do an operation you use **decision-making statements**. 

#### Different types
- If
- If-else
- Switch

**Syntax**

````java
if (condition) {
  // statements
  // we tend to only verify positive outcomes
}
````
```` java
if (condition) {
  // statements
  // we tend to only verify positive outcomes
}
else {
  // statements
}

````
