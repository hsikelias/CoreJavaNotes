## What is an operator?
A operator is a symbol that performs operations on variables and values. 

## Different types of operators in java

These are the major operators
* Arithmetic Operators
  * "+", -, *, / etc
* Unary Operators
* Relational Operators
  * to compare two different values
* Conditional Operators
  * to check something
* Assignment Operators 
  * example: =, this assigns a value

**Arithmetic Operators**:
* "+" Addition
* "-" Subtraction
* "*" Multiplication
* "/" Division
* "%" Modulus

**Unary Operators:**
* ++ Increment ( increases by 1)
* -- Decrement ( decreases by 1)

**Relational Operators:**
* == (equal to)
* != (not equal to)
* > (greater than)
* < (less than)
* >= (greater than or equal to)
* <= (less than or equal to)

**Conditional Operators**

* && AND, || OR

**Assignment Operators**

* =
* +=
* -=
* *=
* /=
* %=
