
```java
package firstPackage;

public class PrintStatements {
  public static void main(String[] args) {
    // Print, Printf, Println
    System.out.print("Hello");
    System.out.print("How are you"); // print prints data in the same line, it ends your cursor position to where it ends
    // not to a new line. 

    System.out.println("Hello"); // moves to a new line, next print happens in a new line
    System.out.printf("How are you %S", "LEKISH" ); 
  }
}

```
Formatting Print Statements

Basically %s means a string, %d means an integer. You can find examples of how it works and more format specifiers here.

The format that you specify (%s) is replaced with the value that you add after the string, for example:

System.out.printf("My name is: %s.", "orexinbaby");  

Would result in this:

My name is orexinbaby
